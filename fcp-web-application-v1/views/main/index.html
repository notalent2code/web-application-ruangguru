About Task Tracker Plus

Task Tracker Plus is the final course project of the Backend Engineering Bootcamp at Ruangguru, designed to assist students in managing their study schedules. Developed using Go programming language, this innovative tool aims to streamline and optimize the study routines of students.

Task Tracker Plus provides a solution for organizing study sessions effectively. With a user-friendly interface and some features, students can easily manage their tasks and optimize their study routines.

Features

  • Add a new task
  • Update tasks that have been added
  • Get task lists and task information
  • Delete unnecessary or already completed tasks
  • Set the category for each task
